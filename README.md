# EditorialAfrica

EditorialAfrica is an AI-powered editorial production platform that uses Google Gemini to generate a complete daily editorial package from leading newspapers in a selected African country.

The MVP is designed around one simple workflow:

> One Country → One Date → One Production Prompt → Complete Editorial Package

---

## Features

- Daily editorial summary
- Comparative editorial analysis
- HeyGen-ready news script
- Social media content (Facebook, LinkedIn, Instagram, TikTok, X)
- Configuration-driven workflow
- Single master production prompt

## Quick Start

1. Update `editorial-date`
2. Verify the country file
3. Open the EditorialAfrica Gemini Gem
4. Run the production prompt
5. Save generated outputs into `artifacts/YYYY-MM-DD`
